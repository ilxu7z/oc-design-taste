# Taste-Skill — 设计品味校准框架

> 基于 Taste-Skill 方法论深度吸收，适配 B 端制造业/画册/官网场景的 AI 设计品味 skill。

## 核心能力

- **三旋钮推理模型** — VARIANCE / MOTION / DENSITY，3 个参数驱动所有设计决策
- **AI 痕迹目录** — 5 大类 30+ 条 AI 常见视觉痕迹，二进制检查
- **Pre-Flight 交付检查清单** — 通用/生图/画册/官网 4 套专项检查
- **子 Agent 任务注入模板** — 派发任务时嵌入不可妥协的设计约束

## 使用

```
npx skills add https://github.com/ilxu7z/Taste-Skill --skill "design-taste"
```
