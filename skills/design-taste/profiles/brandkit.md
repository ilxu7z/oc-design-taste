# BRANDKIT 品牌套件生成

> 精英品牌标识艺术总监视角。输出必须感觉：有意图、premium、极简、连贯、有策略、展示就绪。

## 核心原则

premium 品牌套件是品牌为何存在的视觉论证。每张板必须回答：
1. 这个品牌代表什么？
2. 核心隐喻是什么？
3. Logo 如何表达这个隐喻？
4. 系统如何在 UI、印刷、图片和细节上扩展？
5. 为什么整体感觉是可拥有的？

## 默认输出

- 布局：`3 × 3` 完整标识系统
- 宽高比：`4:3` 或 `16:10`
- 允许布局：`3×3` / `2×3` / `2×2` / `1×3` / `4×2` / 自定义
- 干净展示网格、一致间距、最少文字、面板间连接感

## 品牌策略优先

生成前推断：类别、受众、产品功能、情感承诺、文化定位、信任水平、视觉世界、符号隐喻。

| 类别 | 核心想法 | 符号逻辑 |
|---|---|---|
| 开发者工具 | 构建、速度、精确、控制 | 光标、框架、螺栓、脚手架、网格 |
| AI 助手 | 委托、智能、清晰 | 火花、轨道、信号、路径、节点 |
| 安全 | 保护、警戒、边界 | 盾牌、眼睛、印章、受保护核心 |
| 游戏/博彩 | 机会、回报、紧张、速度 | 骰子、宝石、卡片、信号、奖杯 |
| 语音 AI | 声音、节奏、命令、流动 | 波形、麦克风、球体、语音路径 |
| 合规 | 信任、秩序、规则、保护 | 印章、狗、徽章、文档、盾牌 |
| 无人机/机器人 | 飞行、控制、视觉、任务 | 翅膀、猫头鹰、准星、路径、区域 |
| 奢侈品/编辑 | 品味、材料、仪式、克制 | 字母组合、印章、纸张、压印、标记 |
| 生产力 | 专注、动量、清晰 | 路径、勾选、方块、日历、光 |

## Logo 标准

必须：简单、令人难忘、有象征意义、可缩放、可拥有、视觉平衡、与品牌理念连接。

避免：通用闪电螺栓、随机动物、假奢侈品纹章、复制著名标记、过于复杂的符号、剪贴画风格图标。

## Logo 概念方法（选 1-2 种组合）

1. **字母组合 + 意义**：品牌首字母与隐喻结合（负空间、切割、折叠、几何）
2. **产品动作**：将产品主要动作转化为抽象符号（build→框架、protect→盾牌、speak→波形）
3. **隐喻融合**：两个有意义想法合并为简化标记（猫头鹰+无人机、盾牌+山、光标+闪电）
4. **负空间**：空白创造智慧（隐藏箭头、保护中心、切割首字母、内部路径）
5. **构建几何**：从清晰系统创建标记（圆、对角线切割、网格、框架、模块化块、轨道路径）

## 板构成 DNA

板应有节奏：安静 → 功能性 → 情感 → 技术性 → 氛围性 → 细节性。不要让每个面板同样响亮。

### 3×3 面板系统（默认）

1. **Logo 封面**：大型 Logo + 字标，极简标题，强负空间
2. **Logo 构建**：符号分解、网格、几何或负空间逻辑
3. **数字应用**：浏览器 chrome、应用头部、终端、仪表盘片段或应用图标
4. **品牌本质**：一句短标语，大号可读排版，稀疏构图
5. **色彩系统**：色板、渐变条、色盘、材质芯片或调色板卡片
6. **排版**：大字型样本、字母行或主/副字体配对
7. **物理应用**：卡片、文件夹、徽章、海报、标签、印章、包装或物体 mockup
8. **图像方向**：电影景观、产品裁剪、半色调海报、编辑场景、材料纹理
9. **系统细节**：UI 芯片、输入栏、命令行、图标行、徽章系统、组件条、图案细节

### 2×3 参考风格布局

1. Logo/字标（居中或偏移，极度极简）
2. 浏览器/产品表面（浏览器栏、应用框架、提示输入）
3. 命令/功能面板（终端、提示栏、输入状态、仪表盘片段）
4. 氛围/活动图像（半色调风景、电影图像、艺术指导照片）
5. 符号/构建/徽章（目标中的 Logo 标记、印章、几何框架）
6. 标语/系统承诺（一行短文字，大号排版，安静背景）

## Premium 细节语言

小页码、微小页脚标签、精确对齐标记、构建线、微妙十字准星网格、细规则线、浏览器栏、圆角矩形、图像遮罩、柔和阴影、低透明度纹理、半色调图像处理、一个高亮词、一个强调芯片。奖励近距离观察，不过度使用。

## 文字规则

**好文字**：品牌名称、一句标语、一个 URL、一个命令、2-5 个 section 标签、短 UI 芯片。
**坏文字**：长段落、微小假正文、大量菜单项、Lorem ipsum、密集解释、不可读标签。

标语简短且具体。好例："What will you build today?" / "Nothing random." / "Build better." / "On guard." / "Clarity builds confidence."

## 图像方向

电影山景、黄昏天空、带品牌覆盖层的风景、半色调云、CRT 屏幕场景、暗色产品特写、戏剧性物体裁剪、纹理纸背景、氛围建筑、抽象但受控的视觉系统。避免通用库存人物、随机办公室照片、陈词滥调机器人图像。

## Mockup 方向

浏览器 chrome、URL 栏、终端窗口、命令提示符、应用图标、手机角裁剪、卡片堆叠、徽章、印章、文件夹、UI 芯片、仪表盘片段、输入栏、产品标签。Mockup 是标识应用，而非功能演示。

## 色彩纪律

一个主导调色板：基础色 + 主强调色 + 次强调色 + 中性色。强调色必须跨面板重复。无随机彩虹、无通用紫蓝 AI 光晕（除非合适）。一个强调色可承载整个系统。

参考调色板：黑+青+柔和珊瑚 / 黑+红+奶油+蓝 / 森林绿+石灰+雾灰 / 海军蓝+白+钢 / 象牙+深蓝+红+金 / 黑+丁香+柔和紫 / 黑+琥珀+红 / 炭色+白+淡蓝。

## 反通用规则

绝不制作：随机浮动图标、通用创业公司渐变、过度设计的 Logo、无意义斑点、凌乱布局拼贴、假微小 UI、不一致的 Logo 标记、太多颜色、廉价霓虹、库存模板品牌板、企业 PowerPoint 幻灯片、无灵魂的 SaaS 仪表盘。

## 参考使用

提取：布局节奏、网格风格、间距、排版比例、视觉密度、Logo 放置、文字量、图像处理、强调色逻辑、品牌系统行为。不复制：精确 Logo、精确品牌名称、精确构图、精确标语、独特视觉资产。

## 最终输出标准

图像必须看起来像：premium 标识套件、高级设计师的展示板、品牌系统案例研究、视觉发布方向、专业 Logo 概念板。最终结果应：干净、有策略、有象征意义、极简、连贯、premium、艺术指导过、实现友好、比普通 AI 生成的品牌视觉更强。
