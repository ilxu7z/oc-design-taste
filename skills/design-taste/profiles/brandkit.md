1. 这个品牌代表什么？
2. 核心隐喻是什么？
3. Logo 如何表达这个隐喻？

# Logo 生成标准

Logo 必须是专业的。

它应该：
- 简单
- 令人难忘
- 有象征意义
- 可缩放
- 可拥有
- 视觉平衡
- 与品牌理念连接
- 可用作图标、字标、徽章、UI 标记和图案

避免：
- 通用闪电螺栓（除非有强理由）
- 随机动物
- 假奢侈品纹章
- 复制著名标记
- 过于复杂的符号
- 剪贴画风格图标
- 无意义的火花
- 不一致的 Logo 变体

Logo 应感觉来自研究和提炼。

# Logo 概念方法

使用一种或最多组合两种。

## 1. 字母组合 + 意义

将品牌首字母与隐喻结合。

示例：
- `K` + 风筝 / 框架 / 方向
- `N` + 路径 / 折叠系统
- `S` + 声波 / 语音流
- `A` + 上升 / 建筑 / 动量

不要做一个无聊的字母图标。
使用负空间、切割、折叠或几何。

---

## 2. 产品动作

将产品的主要动作转化为符号。

示例：
- build → 框架、脚手架、块、光标
- protect → 盾牌、边界、看护标记
- convert → 开关、箭头、变换形状
- speak → 波形、麦克风、脉冲
- hunt threats → 眼睛、猛禽、雷达、轨迹
- automate → 循环、交接、路径

使其抽象且 premium，而非字面。

---

## 3. 隐喻融合

将两个有意义的想法合并为一个简化的标记。

示例：
- 猫头鹰 + 无人机视觉
- 盾牌 + 山
- 月亮 + 波形
- 狗 + 合规印章
- 骰子 + 移动游戏经济
- 光标 + 闪电速度
- 风筝 + 产品框架

融合应微妙且可读。

---

## 4. 负空间

使用空白创造智慧。

示例：
- 隐藏箭头
- 保护中心
- 切割首字母
- 内部路径
- 折叠角
- 由交叉形状形成的眼睛

负空间应清晰。

---

## 5. 构建几何

从清晰的系统创建标记。

使用：
- 圆
- 对角线切割
- 网格
- 框架
- 模块化块
- 分层卡片
- 轨道路径
- 十字准星
- 测量线条

一个面板可以展示构建逻辑。

# 2 × 3 参考风格布局

对于类似上传参考的板，使用：

1. **Logo / 字标**
   - 居中或偏移
   - 极度极简

2. **浏览器 / 产品表面**
   - 浏览器栏、应用框架、提示输入或 URL 字段

3. **命令 / 功能面板**
   - 终端、提示栏、输入状态、安装命令、仪表盘片段

4. **氛围 / 活动图像**
   - 半色调风景、电影图像、产品世界视觉或艺术指导照片

5. **符号 / 构建 / 徽章**
   - 目标中的 Logo 标记、印章、几何框架、图标构建

6. **标语 / 系统承诺**
   - 一行短文字
   - 大号排版
   - 安静背景

此布局应感觉像 premium 迷你套件。

# 文字规则

使用非常少的文字。

好文字：
- 品牌名称
- 一句标语
- 一个 URL
- 一个命令
- 2-5 个 section 标签
- 短 UI 芯片

坏文字：
- 长段落
- 微小假正文
- 大量菜单项
- Lorem ipsum
- 密集解释
- 不可读标签

文字应足够大且足够稀疏以良好渲染。

---

# 标语风格

标语应简短且具体。

好：
- "What will you build today?"
- "Nothing random."
- "Your network. Our watch."
- "Build better."
- "On guard."
- "Every mission under control."
- "Everything operators need."
- "Clarity builds confidence."

避免：
- 通用企业口号
- 长营销文案
- 流行词汤
- 假励志废话

# 图像方向

图像应感觉艺术指导过。

使用：
- 电影山景
- 黄昏天空
- 带品牌覆盖层的风景
- 半色调云
- CRT 屏幕场景
- 暗色产品特写
- 戏剧性物体裁剪
- 纹理纸背景
- 氛围建筑
- 抽象但受控的视觉系统

避免：
- 通用库存人物
- 随机办公室照片
- 陈词滥调机器人图像
- 过于繁忙的场景
- 无关图像

图像应匹配调色板和隐喻。

Mockup 应极简且可信。

使用：
- 浏览器 chrome
- URL 栏
- 终端窗口
- 命令提示符
- 应用图标
- 手机角裁剪
- 卡片堆叠
- 徽章
- 印章
- 文件夹
- UI 芯片
- 仪表盘片段

# 色彩纪律

使用一个主导调色板。

默认：
- 基础色
- 主强调色
- 次强调色
- 中性色

好的参考风格调色板：
- 黑 + 青 + 柔和珊瑚
- 黑 + 红 + 奶油 + 蓝
- 森林绿 + 石灰 + 雾灰
- 海军蓝 + 白 + 钢
- 象牙 + 深蓝 + 红 + 金
- 黑 + 丁香 + 柔和紫
- 黑 + 琥珀 + 红
- 炭色 + 白 + 淡蓝

规则：
- 强调色必须跨面板重复
- 无随机彩虹（除非要求）
- 无通用紫蓝 AI 光晕（除非合适）
- 一个强调色可承载整个系统
