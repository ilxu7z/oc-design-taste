# B端印刷品设计规范

> 本领域文件覆盖 B端画册、品牌手册、规格卡、海报、PDF 交付物。
> 所有规则基于 design-taste v4 框架的 Section 2.A（画册/印刷品）美学映射和三旋钮模型。
> 旋钮基线：VARIANCE 3-5 | MOTION 1 | DENSITY 4-6

---

## 1. 印刷品特有规则

### 1.A 色彩模式

| 规则 | 说明 | 突破条件 |
|------|------|---------|
| CMYK 优先 | 所有印刷交付物使用 CMYK 色彩模式 | 数字印刷机支持 RGB 且用户确认 |
| 总墨量 ≤ 300% | CMYK 四色总和不超过 300%，防止背透 | 特种纸张经印刷厂确认 |
| 专色标注 | 潘通色号（Pantone）必须标注在色板旁 | 仅 CMYK 印刷时 |
| 黑色处理 | 大面黑色用 C30 M20 Y20 K100 或 C40 K100（视纸张） | 小字号纯黑文字用 K100 即可 |
| RGB 参考 | 数字预览版可保留 RGB 版本，但交付前标注"预览用" | 最终交付物必须 CMYK |

### 1.B 出血线与安全区

| 规则 | 说明 |
|------|------|
| 出血线 | 四边各 3mm（标准），大幅面海报可用 5mm |
| 安全区 | 四边各 5mm 内不放置关键文字/Logo |
| 背景延伸 | 背景色/图片必须延伸到出血线边缘 |
| 裁切标记 | 最终 PDF 必须包含裁切标记（Crop Marks）和颜色条 |
| 装订预留 | 骑缝钉预留 5mm，胶装预留 3mm 内侧 |

### 1.C 分辨率标准

| 用途 | 最低分辨率 | 说明 |
|------|-----------|------|
| 画册内页 | 300dpi | 标准印刷品质 |
| 封面/精装 | 350dpi | 更高要求 |
| 大幅面海报 | 150-200dpi | 远距离观看可适当降低 |
| 屏幕预览 | 72dpi | 仅用于数字审稿 |

### 1.D 纸张与工艺标注

设计文件中必须标注以下信息供印刷厂参考：
- 纸张类型（铜版纸/哑粉纸/特种纸/牛皮纸）
- 纸张克重（如 157g、200g、300g）
- 表面处理（覆哑膜/覆亮膜/烫金/UV/压纹/击凸）
- 装订方式（骑马钉/胶装/锁线胶装/精装/活页）

## 2. B端画册版式规则

### 2.A 网格系统

| 规则 | 说明 |
|------|------|
| 基础网格 | 12列网格系统，栏间距 4mm-6mm |
| 基线网格 | 正文行高以 4pt 或 5pt 为基线单位对齐 |
| 模块化 | 图片/图表占整数列宽（2/4/6/8/12列），不出现半列 |
| 跨页对齐 | 左右两页的网格基线必须对齐，标题行高一致 |
| 页码位置 | 统一在页面底部外侧（右页右下，左页左下），距裁切边 10mm |

### 2.B 图文比例

| 内容类型 | 图:文比例 | 说明 |
|---------|----------|------|
| 产品展示页 | 7:3 | 大图为主，文字精简 |
| 技术规格页 | 4:6 | 表格/数据为主，配辅助示意图 |
| 品牌故事页 | 6:4 | 氛围大图 + 编辑体文字 |
| 案例页 | 5:5 | 项目图片 + 说明文字均衡 |
| 目录/索引 | 2:8 | 文字为主，小图缩略 |

### 2.C 留白策略

| 区域 | 留白规则 | 说明 |
|------|---------|------|
| 页边距 | 内边 15mm，外边 20mm，天头 15mm，地脚 18mm | 胶装需考虑内边加宽至 20mm |
| 跨页中缝 | 中缝两侧各 10mm 不放置关键内容 | 装订后会夹入书脊 |
| 图文间距 | 图片与文字之间 ≥ 4mm | 防止视觉粘连 |
| Section 间距 | 章节之间留白 ≥ 一个页面（或 1.5 页） | 章节过渡页用整页留白 |

### 2.D 画册结构模板

```
[封面] — 品牌名 + 画册标题 + 年份/版本号
[内封] — 重复封面信息 + 公司全称 + 联系方式
[目录] — 章节标题 + 页码，简洁列表
[品牌序言] — CEO致辞/品牌理念（1-2页，Editorial排版）
[产品总览] — 产品线概览，每类产品 1-2 页
[产品详情] — 每个产品 2-4 页（大图 + 规格 + 卖点）
[技术数据] — 规格对比表、尺寸图、材质说明
[应用案例] — 实拍场景图 + 客户案例
[服务支持] — 售后服务、联系方式、经销商信息
[封底] — 品牌 Logo + 联系方式 + 二维码
```

## 3. YuKoLi 厨具行业特化

### 3.A 品牌理念融入

| 原则 | 执行方式 |
|------|---------|
| 专业厨房基因 | 图片调性偏向商用厨房场景，非家用温馨风格 |
| 德国设计语言 | 排版克制、直角/微圆角、精确对齐、无装饰性元素 |
| 不锈钢质感 | 主色调为冷灰/银灰/不锈钢本色，强调金属反光质感 |
| 功能优先 | 每个页面回答"这个产品解决什么厨房问题" |

### 3.B 产品摄影规则

| 规则 | 说明 |
|------|------|
| 实拍优先 | 优先使用专业产品摄影，次选生图工具生成 |
| 纯白背景 | 产品主图使用纯白/浅灰背景，无阴影干扰 |
| 45度角 | 标准产品展示采用 45度前侧光，展示立体感 |
| 场景图 | 商用厨房实景，包含厨师操作、食材准备等真实场景 |
| 禁止 | 过度饱和、塑料感、无环境上下文的产品渲染图 |
| 尺寸参照 | 产品图旁标注尺寸（长×宽×高 mm），配参照物（如标准锅铲） |

### 3.C 品牌数据引用纪律

YuKoLi 品牌数据引用必须严格基于真实数据，禁止虚构：

| 数据点 | 引用方式 | 示例 |
|--------|---------|------|
| 仓库数量 | "10 大仓库覆盖全国" | 不可夸大或模糊 |
| 发货时效 | "48 小时内发货" | 不可写"24小时"或"极速" |
| 产品线 | 按实际产品目录引用 | 不可写"1000+ SKU"如果实际不符 |
| 客户数量 | 仅引用经确认的数据 | 标注"服务超过 X 家餐饮企业" |
| 质检标准 | 引用实际认证（ISO/CE/FDA 等） | 不可虚构认证 |

**数据标注规则：**
- 真实数据：直接引用，不加修饰
- 估算数据：标注"约"或"超过"
- 未确认数据：标注 [待确认] 并留空
- 禁止使用假精确数字（如"47.2%"如果没有实际统计）

## 4. 品牌手册结构模板

### 4.A 标准品牌手册章节

```
1. 品牌定位（2-4页）
   - 品牌愿景/使命/价值观
   - 品牌调性关键词
   - 品牌故事/起源

2. Logo 使用规范（4-6页）
   - 主 Logo 与变体（横式/竖式/单色/反白）
   - 最小使用尺寸（印刷 ≥ 20mm，数字 ≥ 60px）
   - 安全区（Logo 高度 × 0.5 倍）
   - 禁止使用方式（拉伸/变色/加阴影/旋转）

3. 色彩系统（2-4页）
   - 主色 CMYK + Pantone + HEX + RGB
   - 辅助色
   - 色彩使用比例（主色 60% + 辅助色 30% + 强调色 10%）
   - 禁止色彩组合

4. 字体系统（2-4页）
   - 标题字体 + 正文字体 + 数字字体
   - 字号层级（H1/H2/H3/Body/Caption）
   - 中英文混排规则

5. 品牌图形（2页）
   - 辅助图形/纹理/图案
   - 图标风格规范

6. 摄影风格（2页）
   - 产品摄影规范
   - 场景摄影调性
   - 禁止的摄影风格

7. 应用示例（4-6页）
   - 名片/信纸/信封
   - 画册/产品目录
   - 展会展板/海报
   - 数字应用（官网/社交媒体）
```

### 4.B 品牌手册排版纪律

| 规则 | 说明 |
|------|------|
| 封面 | 品牌色满版 + Logo + 手册标题，无多余信息 |
| 内页页眉 | 左侧品牌 Logo + 右侧章节名，发丝线分隔 |
| 内页页脚 | 页码 + 品牌名 + 版本号（如 V2.0 2025.03） |
| 章节过渡页 | 整页品牌色 + 章节编号 + 章节标题 |
| 禁止 | 纯文字页无视觉元素、Logo 变形使用、色彩溢出安全区 |

## 5. 规格卡排版规范

### 5.A 规格卡结构

```
[头部] — 产品名称 + 产品型号 + 产品图片（小图）
[主体] — 规格参数表（2列或3列表格）
         └ 左列：参数名称（等宽字体，右对齐）
         └ 右列：参数值（比例字体，左对齐）
[底部] — 品牌 Logo + 联系方式 + 二维码 + 版本日期
```

### 5.B 排版规则

| 规则 | 说明 |
|------|------|
| 参数名称 | 等宽字体（JetBrains Mono / IBM Plex Mono），字号 8-9pt |
| 参数值 | 同正文字体，字号 9-10pt，数字用 tabular-nums |
| 表格线 | 仅使用水平细线（0.25pt-0.5pt）分隔行，无垂直边框 |
| 行间距 | 表格行高 5-6mm，确保可读性 |
| 分组 | 相关参数用加粗组标题 + 上方 2mm 间距分组 |
| 单位标注 | 所有数值必须标注单位（mm/kg/℃/V/W），单位字号比数值小 1pt |
| 禁止 | 每行加底线、彩色背景行、装饰性图标在参数旁 |

### 5.C 规格卡尺寸

| 用途 | 尺寸 | 说明 |
|------|------|------|
| 产品随附卡 | 90×55mm（名片尺寸） | 便于放入产品包装 |
| 展示架用卡 | 105×148mm（A6） | 展架/柜台展示 |
| 产品目录规格页 | A4 或 210×285mm | 画册内页 |

## 6. 海报构图规则

### 6.A B端海报构图原则

| 原则 | 说明 |
|------|------|
| 单一视觉焦点 | 一张海报只有一个核心视觉（产品图/场景图/数据图） |
| 视觉层级 | 视觉焦点 → 标题 → 副标题 → 数据/CTA → 品牌信息 |
| 极简文案 | 标题 ≤ 12 字，副标题 ≤ 20 字，正文 ≤ 50 字 |
| 品牌色块 | 使用品牌色作为背景色块或强调色块，面积 ≥ 30% |
| 留白 | 四边留白 ≥ 15mm，元素之间留白 ≥ 5mm |

### 6.B 常见海报版式

| 版式 | 适用场景 | 布局特征 |
|------|---------|---------|
| 产品居中 | 新品发布/主推产品 | 产品图居中，标题在上方，参数在下方 |
| 分屏式 | 对比展示/解决方案 | 左半图片 + 右半文字，或上图片 + 下文字 |
| 大字报 | 促销/活动 | 大号数字/标题占 70% 面积，小图在角落 |
| 网格式 | 多产品展示 | 2×2 或 3×2 网格，每个格子一个产品+名称 |
| 留白式 | 品牌形象 | 极简构图，大量留白，小字品牌信息 |

### 6.C 海报尺寸标准

| 用途 | 尺寸（mm） | 比例 |
|------|-----------|------|
| A3 海报 | 297×420 | 1:1.414 |
| A2 海报 | 420×594 | 1:1.414 |
| 易拉宝/X展架 | 800×2000 | 1:2.5 |
| 展会背景板 | 2000×3000（按实际） | 按展位尺寸 |
| 灯箱 | 按实际尺寸 | 注意光源方向 |

## 7. PDF 交付检查清单

### 7.A 文件设置

- [ ] 色彩模式：CMYK（Coated FOGRA39 或 ISO Coated v2）
- [ ] 分辨率：图片 ≥ 300dpi，灰度图 ≥ 600dpi
- [ ] 出血线：四边 3mm
- [ ] 裁切标记：包含裁切线 + 颜色条 + 色标
- [ ] 字体：所有文字已转曲（Outline）或嵌入字体子集
- [ ] 叠印：黑色文字设置为叠印（Overprint）
- [ ] 透明度：拼合透明度（Flatten Transparency），避免透明叠加问题
- [ ] 文件版本：PDF/X-1a:2001 或 PDF/X-4:2010

### 7.B 内容检查

- [ ] 所有图片已链接（非嵌入预览），且路径有效
- [ ] 无 RGB 图片残留
- [ ] 无缺失字体
- [ ] 无超链接（印刷品不需要）
- [ ] 所有文字在安全区内
- [ ] 跨页图片正确对齐中缝
- [ ] 页码连续且正确
- [ ] 封面封底在同一个 PDF 文件内（或按印刷厂要求分文件）

### 7.C 品牌一致性检查

- [ ] Logo 使用符合品牌手册规范
- [ ] 品牌色 CMYK 值正确
- [ ] 字体使用统一（同一层级用同一字号字重）
- [ ] 所有数据引用真实可查
- [ ] 无 em-dash 在任何可见文本中
- [ ] 无 AI 痕迹文案（假精确数字、假谦虚行业引用、感叹号滥用）

### 7.D 交付物清单

| 文件 | 格式 | 说明 |
|------|------|------|
| 印刷用 PDF | PDF/X-4 | 含出血线、裁切标记，分页 |
| 跨页 PDF | PDF/X-4 | 跨页拼版版本，用于审稿 |
| 屏幕预览 PDF | 压缩 PDF | 低分辨率（72dpi），用于邮件发送 |
| 源文件 | AI/INDD/AFFINITY | 按印刷厂要求提供 |
| 字体包 | OTF/TTF | 嵌入字体文件（如未转曲） |
| 图片包 | TIFF/PSD | 原始高分辨率图片（按需） |

## 8. 印刷品特有 AI 痕迹警告

以下模式在 B端印刷品中尤其常见，必须主动检测并修复：

| 痕迹 | 表现 | 修复 |
|------|------|------|
| 假数据表格 | 用"XX"或"--"填充的规格表 | 留空或标注"待确认" |
| 过度渲染图 | 产品图无尘无指纹，像 CG 渲染 | 使用实拍或 prompt 加"workshop feel" |
| 假证书图标 | 虚构的认证/专利图标 | 仅使用真实认证 Logo |
| 文案空洞 | "卓越品质""行业领先"等空话 | 替换为具体参数和功能描述 |
| 忽略装订 | 中缝放置文字/人脸 | 中缝两侧各 10mm 安全区 |
| 色彩不一致 | 同一品牌色在不同页面有色差 | 锁定 CMYK 值，禁止视觉调色 |
| 假页码 | 页码从"02"开始或跳过封面 | 封面不计页码，内封为 p1 |
| 过度装饰 | 装饰线/花纹/底纹过多 | B端印刷品以功能为主，装饰克制 |

## 9. 三旋钮映射（印刷品特化）

| 旋钮 | 画册 | 品牌手册 | 规格卡 | 海报 |
|------|------|---------|--------|------|
| VARIANCE | 3-5 | 4-6 | 2-3 | 5-7 |
| MOTION | 1 | 1 | 1 | 1 |
| DENSITY | 4-6 | 3-5 | 7-9 | 2-4 |

**说明：** 所有印刷品 MOTION 固定为 1（静态）。DENSITY 根据内容类型调整——规格卡最密，海报最疏。
